================================================================================
                      5-MINUTE QUICK START
================================================================================

Step 1: Install (2 min)
  cargo install cargo-dist --locked

Step 2: Initialize (2 min)
  cargo dist init
  (Accept defaults when prompted)

Step 3: Test (1 min)
  cargo dist plan
  (See what will be built)

That's it! You now have:
  - Configured cargo-dist for your project
  - Generated GitHub Actions release workflow
  - Ready to tag and release

To release:
  git tag v1.0.0
  git push --tags
  (Let CI build and publish everything!)

================================================================================
                        KEY FACTS AT A GLANCE
================================================================================

What: Automated release tool for Rust binaries
Why: Eliminates manual release process complexity
How: Run 'cargo dist init' once, then 'git tag && git push'
Who: Maintained by axodotdev, 76+ contributors
When: Version 0.30.2 (December 2025)
Where: Works with GitHub Actions (GitLab coming soon)

Supports:
  - Linux (x86-64, ARM, PowerPC, RISC-V, s390x variants)
  - macOS (Intel & ARM64)
  - Windows (x86-64 & ARM64)
  - 14+ target triples total

Features:
  - Automatic cross-platform builds
  - Multiple installer types
  - GitHub release integration
  - Checksum generation
  - GitHub Attestations for security
  - Self-hosted proof of stability
